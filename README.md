# WarikanApp

ユーザとユーザの支払い立替え情報を入力することで、誰が誰にいくら返せばいいか計算してくれるアプリです。

## インストール手順
1. リポジトリをクローン
   ```sh
   git clone https://github.com/ryu-dra/WarikanApp.git
   cd WarikanApp
2.Android Studio で開く
3.Run ボタンでアプリを実行

## 主な機能
ユーザー情報の登録・編集
立て替え情報の登録
返済方法の表示

## 使用技術
言語: Kotlin
UI: Jetpack Compose
データベース: Room
